X-Ray Information System: An Image Processing and Deep Neural Network Approach

BSc (Hons) Final Year Research Project

üß† Overview: This project aimed to develop an automated diagnostic system that detects bone fractures from musculoskeletal X-ray images using deep learning techniques. By leveraging the MURA v1.1 dataset, which includes over 40,000 radiographs across seven body parts (elbow, finger, forearm, hand, humerus, shoulder, and wrist), the system was trained to identify the body part and determine whether a fracture is present

üí° Key Contributions:

  • Developed a two-stage deep learning pipeline using ResNet50: the first stage for body part classification, and the second for fracture detection.
  • Used transfer learning and fine-tuning strategies to improve accuracy on medical imagery.
  • Achieved 83% ,81% testing accuracy for body part detection and fracture detection models respectively with strong precision, recall, and F1-score metrics.
  • Implemented image preprocessing techniques and data balancing to enhance model robustness.
  • Integrated the trained model into a Flask-based API and connected it with a React Native mobile application for real-time diagnosis.
  • Built a user-friendly interface enabling doctors and patients to upload X-rays and receive instant, reliable fracture predictions.


üì± Outcome

The final solution offers a fast, accurate, and accessible fracture detection tool that can assist healthcare professionals in preliminary diagnostics, particularly in remote or resource-limited settings.
